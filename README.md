## Tech Stack
- Frontend: `React`
- Server: `Express`
- Database: `MongoDB`
- Auth: `JWT`

## Installation & Setup

### Requirements
  -  Node.js: `^22.14.0`

```bash
# 1. Clone the repo
git clone https://github.com/quantum2code/habit-tracker

cd habit-tracker

# 2. Install dependencies
npm install
```

## How to Run
 ```bash
 # Dev mode
 # 1. web client
 npm run dev:web

 # 2. server
 npm run dev:server
 ```
